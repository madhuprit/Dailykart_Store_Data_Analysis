# Dailykart_Store_Data_Analysis

Excel Data Analysis Project | white label store

Data Management: Cleaned and formatted complex sales datasets using advanced Excel functions (Text-to-Columns, Filters) to ensure data integrity.

Segmentation: Engineered a new "Age Group" variable to categorize customers, revealing that the Adult segment is the highest revenue contributor.

Dashboarding: Designed an interactive dashboard with Slicers and Pivot Charts to visualize monthly trends, seasonality, and regional performance.

Strategic Insights: Identified that Women are the top demographic and Myntra/Amazon are the dominant sales channels, with high demand concentrated in MH, KA, and UP.

Outcome: Successfully translated transactional data into a visual story that supports strategic decision-making.
